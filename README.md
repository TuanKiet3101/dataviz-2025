# Salary Survey Data Cleaning Project

## Project Overview
This project performs end-to-end data cleaning
and preprocessing for a salary survey dataset.

## Files
- 01_salary_cleaning.ipynb
- 02_data_quality_report.md
- 03_salary_cleaned.csv
- 04_README.md

## Main Problems Handled
- Missing values
- Duplicate rows
- Wrong data types
- Outliers
- Inconsistent text values
- Encoding categorical variables

## Techniques Used
- IQR
- Z-score
- Winsorization
- One-Hot Encoding
- Feature Engineering
- MinMaxScaler
- StandardScaler

## How to Run
1. Open notebook
2. Run all cells
3. Export cleaned dataset

## Final Result
Dataset was cleaned and transformed into
analysis-ready format.
